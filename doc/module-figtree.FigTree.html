Class: FigTree

figtree.FigTree(svgopt, margins, tree, settings) → {FigTree}

The FigTree class A class that takes a tree and draws it into the the given SVG root element. Has a range of methods for adding interactivity to the tree (e.g., mouse-over labels, rotating nodes and rerooting on branches). The figure updates with animated transitions when the tree is updated.
