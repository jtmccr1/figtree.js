(constant) equalAngleLayout

The equal angle layout. This function returns a layout function. It take and optional internal node, which if provided acts as the starting node and fixes the order nodes are visited. This means the tree not update. The root position will still change in response to rerooting.

(constant) rectangularHilightedLayout

This layout highlights parts of the tree and compresses others. The layout factory takes a predicate function that is called on each node and returns true or false. true == more space around node.
