The layout function is called to make the edges, vertrices, and cartoons that make up the graph and set or update their positions on a custom x and y scale. It also should update the _horizontialScale, the _veriticalRange so that these positions can be converted to a [0,1] scale to be consumed by the figtree class.

setXPosition(vertex, currentX) → {number}

Set the x position of a vertex and return the X position. This function is called on each node in the order returns from the getTreeNodes() method. The currentX represent the x position of the previous node at each iteration. These x values will be mapped to a [0,1] range. In the 'normal' left to right tree this method would ignore the currentX and set the x based on the horizontal scale.
